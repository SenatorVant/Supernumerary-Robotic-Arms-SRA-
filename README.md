# SRA Project – Supernumerary Robotic Arms

## Overview
The Supernumerary Robotic Arms (SRA) Project focuses on the design and development of wearable robotic arms that augment human capability. These additional robotic limbs operate alongside the user’s natural arms, enabling enhanced multitasking, strength, and precision in complex tasks.

This project combines mechanical design, embedded systems, and intelligent control to create a seamless human-robot interaction system.

## Objective
The goal of this project is to:
- Develop a functional wearable robotic arm system  
- Enable intuitive and responsive human control  
- Augment human ability without restricting natural motion  
- Explore real-world applications of human augmentation technology  

## Features
- Wearable multi-DOF robotic arm system  
- Real-time control using sensors and/or custom input methods  
- Lightweight and ergonomic mechanical design  
- Modular architecture for scalability and upgrades  
- Integrated control system for smooth and precise motion  

## Tech Stack
- **Programming:** Python / C++ / Arduino  
- **Electronics:** Microcontrollers, motor drivers, sensors  
- **Mechanical Design:** Fusion 360 / SolidWorks  
- **Manufacturing:** FDM / SLA 3D printing, machining  
- **Control Systems:** PID control, kinematics modeling  

## System Architecture
1. **Input Layer:** User input via sensors (IMU, EMG, buttons, or gesture-based systems)  
2. **Processing Layer:** Microcontroller processes signals and maps them to motion commands  
3. **Control Layer:** Algorithms regulate movement (position, speed, torque)  
4. **Actuation Layer:** Motors drive the robotic arms  
5. **Output:** Physical augmentation of user capability  

## Applications
- Industrial assistance and manufacturing  
- Medical rehabilitation and assistive devices  
- Robotics and human augmentation research  
- Task augmentation (holding, stabilizing, multitasking)  

## Challenges
- Achieving intuitive and low-latency control  
- Maintaining lightweight yet strong structures  
- Power efficiency and battery integration  
- Ensuring user safety and comfort  

## Results
- Functional prototype demonstrating coordinated robotic arm movement  
- Successful integration of mechanical, electrical, and software systems  
- Improved task efficiency through augmentation testing  

## Future Improvements
- Integration of EMG-based neural control  
- AI-assisted motion prediction and automation  
- Enhanced ergonomics and weight distribution  
- Increased degrees of freedom and precision  
- Wireless communication and compact power systems
