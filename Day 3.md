# Final Gearbox Architecture

The gearbox is a **bus servo driving a reduction system that outputs to a ring gear** at the arm joint. This setup allows the servo to generate significantly more torque than it could before.

#### The system works as follows:

- A **bus servo** provides the primary actuation
- The servo drives a **gear reduction stage**
- The reduction stage transfers motion into a **ring gear**
- The **ring gear rotates the arm joint**

This configuration allows for **high torque, smooth motion, and compact packaging**, which is important for a wearable robotics system.

# Design Considerations

While finishing the CAD, I focused on several important engineering considerations.

**1. Strength**

The gearbox needs to handle the torque generated when lifting or stabilizing objects.

- Thickened structural areas of the housing  
- Reinforced mounting points  
- Proper gear engagement depth
- Usage of different materials to maximize strenth

**2. Weight**

Because the arms will be worn on the back, minimizing weight is important.

- Reduced unnecessary material in the housing  
- Simplified geometry where possible  
- Using materials with low density such as abs will be required

**3. Modularity**

The gearbox was designed so it can be easily modified or upgraded later.

- Servos can be swapped if higher torque is needed  
- Gear ratios can be adjusted in future versions  
- Housing can be reprinted if improvements are made
